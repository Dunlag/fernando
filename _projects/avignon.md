## Finca Avignon — Landing Page para Promoción Inmobiliaria

Proyecto web desarrollado en **agencia** para cliente real. Landing de presentación y captación de contactos para una promoción de apartamentos, actualmente **en producción** en [fincaavignon.es](https://fincaavignon.es/).

![Screenshot Finca Avignon]({{ '/assets/images/projects/avignon.png' | relative_url }})

### Contexto del proyecto

Promoción inmobiliaria que necesitaba presencia digital orientada a la captación: presentación visual de los apartamentos, galería de imágenes y un formulario de contacto funcional para gestionar el interés de compradores.

### Mi rol

- 🎨 **Diseño visual** alineado con la identidad de la promoción
- 🖥️ **Maquetación responsive** mobile-first para cualquier dispositivo
- 📬 **Formulario de captación de leads** con lógica PHP y envío por correo
- 🖼️ **Galería de imágenes** de viviendas y entorno optimizada para carga rápida
- 🔄 **Iteración con cliente** — revisiones sobre feedback real hasta aprobación final

### Proceso

| Fase | Descripción |
|---|---|
| **Brief** | Reunión con cliente, análisis de requisitos y público objetivo |
| **Diseño** | Propuesta visual adaptada a la identidad de la promoción |
| **Desarrollo** | Maquetación frontend + formulario PHP |
| **Revisión** | Iteración con cliente hasta aprobación final |
| **Entrega** | Despliegue en producción |

### Stack técnico

| Herramienta | Uso |
|---|---|
| **PHP** | Backend, gestión de formularios, envío de correo |
| **HTML5** | Estructura semántica y accesible |
| **CSS3** | Diseño responsive, animaciones, identidad visual |
| **JavaScript** | Interactividad y mejora de experiencia de usuario |
