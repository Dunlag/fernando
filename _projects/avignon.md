## Finca Avignon

Desarrollo completo de la página web oficial de **Finca Avignon**, proyecto real para cliente que muestra las instalaciones, servicios y permite gestionar reservas o consultas de contacto.

### Características

- 🏡 **Diseño elegante** que refleja la identidad y ambiente de la finca
- 📱 **Totalmente responsive** para cualquier dispositivo
- 📬 **Formulario de contacto/reservas** con gestión en PHP
- 🖼️ **Galería de imágenes** de instalaciones y entorno
- 🗺️ **Información de localización** y accesos
- ⚡ **Rendimiento optimizado** con carga eficiente de recursos

### Tecnologías Utilizadas

- **PHP** para lógica de servidor, envío de formularios y gestión de contenido
- **HTML5** semántico con estructura accesible
- **CSS3** con diseño responsive y estética cuidada
- **JavaScript** para interactividad y experiencia de usuario

### Tipo de Proyecto

Proyecto profesional desarrollado para cliente real. Web corporativa con backend PHP para gestión de contacto y presentación de servicios.
